# 📊 Sales Forecasting & Demand Prediction Analysis

# 📌 Project Overview

This project focuses on analyzing historical retail sales data to identify trends, seasonal patterns, and forecast future demand. The goal is to support better business decisions related to inventory management, sales planning, and demand forecasting.

# 🎯 Objectives
* Analyze historical sales data
* Identify monthly and yearly sales trends
* Detect seasonal demand patterns
* Forecast future sales
* Build an interactive dashboard for business insights
# 📂 Dataset
* Generated dataset with 12,000+ records
## Features include:
* Date
* Product ID
* Product Category
* Region
* Units Sold
* Price
* Revenue
# 🧹 Data Cleaning & Preprocessing
* Converted Date column to datetime format
* Removed duplicate records
* Checked and handled missing values
#### Created new features:
*  Year
*  Month
* Month Name
# 📊 Exploratory Data Analysis
#### 🔹 Sales Trend

#### 🔹 Seasonal Patterns

#### 🔹 Product Performance

#### 🔹 Regional Analysis
